# dyn_ai — Live AI Tuner for GTR2 - v1.1.0

Dyn AI detects qualifying and race results and tunes AI to match your pace.

1. Download dyn_ai.zip and dyn_ai.z01
2. Uncompress to a folder of your liking. Get in that folder.
3. Run dyn_ai.exe, follow instructions (set GTR2 installation path, import vehicles...).
4. Leave it running and boot GTR2.

The program will then automatically adjust GTR2 AI difficulty (QualRatio/RaceRatio) for each Track, based on your lap times.
Setup and graph tools are also available (dyn_ai_setup.exe, dyn_ai_visualizer.exe).

## Tips

- The more races with a given car on a given track, the more precise the program will be.
- Once you like the AI level for that track and car, you can disable "Auto-calculate Ratios" at the bottom.
- If you want to understand how the Ratios are calculated, you can check the data using dyn_ai_visualizer.
  - Moreover, here you can manually change the formulas, the latest User Laptime... Use with caution.
- If you want to correct a Ratio by hand, turn off the "Auto-calculate Ratios" and click on "Edit".
